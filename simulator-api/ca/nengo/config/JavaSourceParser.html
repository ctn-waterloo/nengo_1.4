getMethod¶

public static Method getMethod(String reference, String referringClassName)¶
Parameters
  • reference – A JavaDoc see-tag-style reference, ie fully.qualified.ClassName#methodName(ArgType0, ArgType1)
  • referringClassName – Name of class on which the see tag is written (used to find default package if arg types are not qualified, and class if undefined)
Throws
  • SecurityException – if we can’t access the method
  • NoSuchMethodException – if the method doesn’t exist
  • ClassNotFoundException – if the class doesn’t exist
Returns

Matching Method if possible, otherwise null
