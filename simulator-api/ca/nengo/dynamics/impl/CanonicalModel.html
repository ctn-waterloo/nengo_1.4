getRealization¶

public static LTISystem getRealization(float[] numerator, float[] denominator, float passthrough)¶

Realizes a transfer function in the form:

H(s) = d + (b1*s^(n-1) + b2*s^(n-2) + … + bn) / (s^n + a1*s^(n-1) + … + an).

Parameters
  • numerator – Coefficients of the numerator of a transfer function (b1 to bn above)
  • denominator – Coefficients of the denominator of a transfer function (a1 to an above)
  • passthrough – Passthrough value (d above). If your transfer function has numerator and denominator of equal degree, divide them, give the result here, and give the remainder as the numerator and denominator arguments, so that the new numerator will have degree less than denominator. There is no state-space realization for TF with numerator degree > denominator degree.
Returns

A controllable-canonical state-space realization of the specified transfer function
