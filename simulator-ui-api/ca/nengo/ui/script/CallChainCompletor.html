CallChainCompletor¶

public class CallChainCompletor extends CommandCompletor¶

A CommandCompletor that suggests completions based on Python variable names and methods/fields of Python objects.

Author
Bryan Tripp

getMembers¶

public List<String> getMembers(String base)¶
Parameters
  • base – A variable name in the interpreter. For variables that wrap Java objects, this arg can consist of a call chain, eg x.getY().getZ() For native python variables, the return type of a call isn’t known, so this method can’t return anything given a call chain.
Returns

A list of completion options including methods (with args) and fields on the named variable.

setBase¶

public void setBase(String callChain)¶

Rebuilds the completion options list from a “base” call chain.

Parameters
  • callChain – A partial call chain, eg “x.getY().get”, from which we would extract the base call chain “x.getY()”. (In this case the new options list might include “x.getY().toString()”, “x.getY().wait()”, etc.)
